# Release v0.21.2

A quality-of-life release for the terminal app's two input surfaces. The file
editor learns to take the mouse — a trackpad or mouse left click now places
the cursor exactly where you click — and the AI prompt starts behaving like a
real editor: long prompts soft-wrap instead of running off-screen, ↑/↓ walk
the input by visual row, and Ctrl+A grows a progressive selection you can
replace, delete, or dismiss.

- **Left click places the edit cursor.** In edit mode, clicking a cell moves
  the cursor there — through the same clamps as the arrow keys, so a click on
  the line-number gutter snaps to column 0, a click past the end of a line
  lands at its end, and clicks on the sidebar, header, or the blank pane below
  the last line do nothing. Drags, releases, right clicks, and stray trackpad
  gestures are deliberately inert. Mouse capture is enabled only while
  editing, so native terminal text selection and copy keep working in the
  query, view, and history panes.
- **The AI input soft-wraps.** A prompt longer than the terminal is wide used
  to overflow its row; now it wraps at the window width and grows downward,
  with continuation rows aligned under the `@ai >` prompt. The cursor is
  always visible — one cell is reserved at the end of each row so the
  terminal never auto-wraps behind the renderer's back.
- **AI input keys reworked.** Plain ↑/↓ now move the input cursor one visual
  row (top row jumps to the start, bottom row to the end), so editing a
  wrapped prompt feels like a tiny editor; transcript scrolling moves to
  Shift (or Option) + ↑/↓, and the hint row says so. Forward Delete works.
  Ctrl+A selects progressively — first press takes the cursor's line, the
  second (or an empty line) the whole input — rendered highlighted across
  wrapped rows: Backspace/Delete removes the selection, typing replaces it,
  Esc keeps the text and just deselects.
- **Notices surface in AI mode.** Transient notices like `@copy`'s "copied"
  now ride the hint row beneath the AI input, so they are visible without
  costing a screen row; any keystroke clears them.
