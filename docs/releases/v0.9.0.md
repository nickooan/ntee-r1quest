# Release v0.9.0

Adds first-class GraphQL authoring support, including `.ntd` operation
shorthand, GraphQL-aware highlighting, runnable examples, and AI skills for
GraphQL request generation.

- Added top-level GraphQL operation shorthand in `.ntd` definition files, so
  `query ... { ... }` compiles to the `query` key and `mutation ... { ... }`
  compiles to the `mutation` key.
- Kept existing quoted `query: "..."` and `mutation: "..."` definition values
  compatible for explicit string-based GraphQL operation storage.
- Added compiler and grammar coverage for GraphQL query and mutation shorthand
  in definition files.
- Added GraphQL-aware syntax highlighting for query and mutation definition
  values in the terminal file view and edit view.
- Updated GraphQL highlighting to support both quoted operation values and the
  new top-level shorthand blocks, with lightweight brace-depth detection.
- Added GraphQLZero examples with separate `graphql/` operation `.ntd` files
  and executable request files under `request/queries/` and
  `request/mutations/`.
- Added runnable query, nested query, and mutation examples for GraphQLZero,
  including `query-post`, `query-user`, `query-user-posts`,
  `query-album-photos`, and `mutation-create-post`.
- Verified the GraphQLZero examples compile and run through one-shot `-p`
  execution.
- Added `r1quest-graphql-generator` AI skill guidance for creating GraphQL
  `.ntd` operation/variables files and matching `.nts` request actions.
- Added `graphql-schema-r1quest-generator` AI skill guidance for generating a
  R1Quest GraphQL project from schema files, including schema root handling,
  `type`, `input`, `enum`, `interface`, `union`, and custom scalar support.
- Updated GraphQL AI skill guidance for multiple `.ntd` refs, shared auth/data
  reuse, and later-ref override behavior.
- Updated README with a top-level index, GraphQL request documentation, example
  GraphQLZero file paths, `-p` commands, and multiple-ref guidance.
- Updated plugin documentation to list the new GraphQL-focused skills.
