# Release v0.9.1

Fixes long-running terminal memory growth that could eventually trigger a Node
heap out-of-memory crash after the app sat open for many hours.

- Stopped embedding full AI assistant message content in React render keys for
  wrapped AI chat lines, avoiding repeated large-string allocation during
  terminal re-renders.
- Stopped embedding highlighted source segment text in file-content render keys,
  reducing reconciliation pressure for large viewed or edited files.
- Added an idle timeout for cursor blinking, so the terminal stops rendering
  every 500ms after 30 seconds without input or pending work.
- Restarted cursor blinking automatically on the next keypress and kept the
  cursor visible while idle.
- Memoized file-tree expansion and sidebar entry building so cursor renders do
  not repeatedly rescan and rebuild the request collection.
- Added unit coverage to ensure AI render keys do not include full message
  content.
