# Release v0.15.0

Rewrites the terminal UI in Go (Bubble Tea) talking to the TypeScript runtime
over JSON-RPC, makes it the default — and only — interactive front-end, removes
the legacy Ink UI, and lands a wave of query/history/AI interaction
improvements and fixes.

- **Breaking — new architecture & default UI.** The interactive TUI is now a
  Go / Bubble Tea binary. The TypeScript runtime (parser, cache, HTTP
  execution, AI/ACP adapters) runs as a separate process behind a JSON-RPC
  protocol over a per-run Unix-domain socket. `r1q` launches the Go binary for
  interactive sessions and keeps the one-shot flags (`-p`, `--init`,
  `--version`, `--install-claude-plugin`) in TypeScript.
- **Breaking — the Ink TUI is removed.** The Go binary is the only interactive
  UI; the `R1QUEST_INK` / `--ink` fallback is gone. Platforms without a prebuilt
  binary (Windows, non amd64/arm64) have no interactive UI — use one-shot mode
  (`-p`). The `--init` wizard still runs.
- **Breaking — building from source now requires the Go toolchain (1.24+).**
  `install.sh` checks for Go and Node 24+ up front and builds the UI binary; the
  `build` script cross-compiles all targets.
- Prebuilt Go binaries (macOS/Linux × amd64/arm64) are cross-compiled into
  `dist/` and shipped in the npm package. The Claude plugin marketplace is now
  bundled in `dist/` too, so `--install-claude-plugin` works for npm installs.
- **Breaking — query keybindings changed.** Up/Down now scroll the result pane
  (Shift+Up/Down move the sidebar selection), and `s` opens search (was `/`).
  View, edit, search, and history exit with Esc.
- Query mode: typing shows a suggestion popup mixing current-directory files
  with previously run inputs (green); Shift+Up/Down selection mirrors into the
  input bar and popup; Enter acts by type — enter a directory, run a `.nts`
  request, or confirm-to-view any other file. A trailing `<path> @v` / `@e`
  opens a file directly, and binary files show a "not readable" overlay.
- History mode: Shift+Up/Down select an entry, Up/Down scroll the selected
  record, `s` searches it, and `@h <traceId>` filters to a trace's calls in
  order. The query result now also shows request duration after the status
  (`404 Not Found · 680 ms`), and an external one-shot result highlights its
  originating request in the sidebar.
- AI mode: a session picker on entry (resume a past session or start new),
  multi-line input (Ctrl+J newline; Shift/Opt+Up/Down move the cursor), an
  animated "<Agent> is thinking…" indicator, and a `/command` popup for custom
  commands. `@` commands typed in chat run as TUI actions instead of being sent
  to the agent, and each response turn renders on its own line.
- Fixed out-of-order AI streaming: JSON-RPC notifications are dispatched in
  arrival order, so message chunks no longer interleave or corrupt.
- `@reload` re-resolves config and resets the UI (and any live AI session);
  `@clean-cache` / `@cc` clears input and request history.
- Removed the legacy Ink view layer and the runtime helpers only it used
  (~60 modules) now that the Go UI is at parity.
